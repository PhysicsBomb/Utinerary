Utinerary

Application to streamline your trips!

View project on GitHub

Thanks for visiting! Currently, Utinerary can be downloaded. The build is stable but is still a WIP (Work-In-Progress. If you want to see it so far, download the .zip file. After unzipping it, open the directory Youtinary-1 and run the HTML File revf.html with Google Chrome Stable or any other stable browser other than Microsoft Edge.

Utinerary is an application that simplifies trying to find something to do when on vacation, or trying to find something to eat. It allows you to do what used to take many applications with only one application.*

If you see any visible bugs or have any feedback, please e-mail csobhanachala@gmail.com, and I will try my best to rectify the problem as soon as possible.

*Contingent on Internet Connection.
